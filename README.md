# AI Study Lab

AIを活用した学習教材の開発・公開ラボ。HTMLベースの学習教材を試作・公開しています。

🌐 **公開サイト**: https://ai-study-lab.pages.dev (Cloudflare Pages 連携後に有効化)

## 何をしているか

- 生成AIを使った学習教材の試作
- 業務効率化・資格学習・スキル習得をテーマに開発
- 開発過程と完成品を発信

## ディレクトリ構成

各教材は独立したフォルダで管理：

```
lab/
├── index.html       # 教材一覧トップ
└── [教材名]/        # 各教材フォルダ
    └── index.html
```

## 開発について

- 作業リポジトリ（Private）で試作 → このPublicリポに完成品をコピー公開
- 詳しい運用は astranode01 の管理ドキュメント参照
